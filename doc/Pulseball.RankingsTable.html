Class: RankingsTable

PULSEBALL. RankingsTable

new RankingsTable(rankingsJson)

This class represent a rankings table. A rankings table is a set of rows. Each rows contains information about a team and its position in the tournment. In this implementation, the rows of the table are ordered by score value (descendent).
